# **🍔 FoodExpress - Online Food Ordering Website - A Simple frontend project**
FoodExpress is a responsive frontend food ordering web application built using HTML, CSS, and JavaScript. It allows users to browse food items, filter by categories, add items to a cart, and simulate checkout.

This project demonstrates DOM manipulation, responsive UI design, and interactive cart functionality using vanilla JavaScript.

# **Features**
📋 Browse food menu items 🍕 Filter foods by category (Pizza, Burger, Pasta, Salad, Dessert) 🛒 Add items to cart ✅ Button changes to "Added to Cart" with green color ➕ Increase / ➖ decrease item quantity 🗑 Remove items from cart 💰 Automatic total price calculation 🛍 Cart modal popup 📱 Responsive design for mobile devices 🔄 Buttons reset after checkout

# **🛠 Technologies Used**
HTML5 – Structure of the application CSS3 – Styling and responsive design JavaScript (Vanilla JS) – Dynamic functionality and DOM manipulation Bootstrap - interactive UI sections Font Awesome – Icons

 # **📂 Project Structure**
FoodExpress/ │ ├── index.html # Main webpage ├── food.css # Styling and responsive design ├── food.js # Application logic └── README.md # Project documentation

# **⚙️ How It Works**
The menu items are stored in a JavaScript array. JavaScript dynamically renders the menu items on the webpage. When a user clicks Add to Cart: The item is added to the cart array. Button changes to Added to Cart.

# **The cart modal displays:**
Items Quantity controls Total price When the user clicks Checkout: An order confirmation appears Cart resets Buttons revert to Add to Cart

# **▶️ How to Run the Project**
Clone the repository Open the project folder Run the project by opening index.html in your browser. No installation or backend required.
